New version 4.0

  • A registration system with many functions, including password recovery or "Keep login" was added
  • A footer referring to GitHub page has been added.
  • Average number of HTTP requests per minute averaged to statistical groups of fifteen minutes is now being written to the database.
  • Error pages have been remade and they provide more information, for example recommended actions.
  • When a session expires, users are redirected back to the list of recognition tests.
  • The icon of the web was changed.
  • Changelog was changed.
  • Slight changes in colour pallet were made.
  • The form for adding pictures doesn't reset entirely now. Instead, the chosen organism stays the same, which makes adding more pictures of the same organism faster.
  • The numbers in brackets at "Adding pictures" pages are now being refreshed right after successful addition.
  • Mecessity to enter access code and functions related to it were removed.
  • Forgotten console reports used for developement purposes have been removed.
